# 🛡️ DriveGuard

## AI-Based Driver Drowsiness Monitoring and Alert Management System

DriveGuard is an AI-powered web application designed to improve road safety by detecting driver drowsiness in real time using Computer Vision and Artificial Intelligence. The system continuously monitors the driver's alertness through a webcam and provides instant safety alerts and recommendations to help prevent fatigue-related road accidents.

---

## 🚀 Project Overview

Driver fatigue is one of the leading causes of road accidents worldwide. DriveGuard aims to reduce these risks by monitoring the driver's facial features and eye movements using AI-based computer vision techniques.

When signs of drowsiness are detected, the system immediately alerts the driver and provides safety recommendations such as taking a break, drinking water, or having coffee.
